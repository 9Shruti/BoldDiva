# BoldDiva

## Overview
BoldDiva is a beauty and cosmetics e-commerce website developed using Django, HTML, CSS, JavaScript, and SQLite.

## Features
- User Registration & Login
- Product Catalog
- Shopping Cart
- Checkout
- Admin Dashboard
- Order Management
- Responsive Design

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
